**Aphid** is the build log and research notebook for converting a 1961 VW
Type 1 (Beetle) into an electric vehicle using a Nissan Leaf drivetrain.

The Swedish name for an aphid is *bladlus* — leaf louse — which felt like
the right name for a Leaf-powered Bug.

Follow the **blog** for project updates as the build progresses, or dig
through the **wiki** for component teardowns, pinouts, and reference notes.
